jQuery Monta-method plugin

This is a Monta-method plugin for jQuery.

Usage

Let's click blinded texts!
  1. Include jquery (1.10.1 tested) 
    <script src="http://code.jquery.com/jquery-1.10.1.min.js"></script>
  2. Include jquery-monta.js 
    <script src="./jquery-monta.js"></script>
  3. Call monta() method for jQuery elements. 
    <script>
  $('span').monta();
</script>
